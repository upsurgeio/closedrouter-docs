# Models

ClosedRouter gives you access to **two major model providers** under one API key:

- **OpenAI Models** : GPT family optimized for general-purpose reasoning, code generation, and structured outputs
- **Anthropic Models** : Claude family optimized for long context, safe alignment, and balanced reasoning

## ðŸ”§ API Usage Examples

### Basic Chat Request

```bash
curl https://api.closedrouter.io/v1/chat/openai \
  -H "Authorization: Bearer $CLOSEDROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "Hello, world!"}
    ]
  }'
```

### Anthropic Chat Request

```bash
curl https://api.closedrouter.io/v1/chat/anthropic \
  -H "Authorization: Bearer $CLOSEDROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-3-sonnet",
    "messages": [
      {"role": "user", "content": "Explain quantum computing in simple terms"}
    ]
  }'
```

### JavaScript Example

```javascript
const response = await fetch('https://api.closedrouter.io/v1/chat/openai', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4o',
    messages: [
      { role: 'user', content: 'Write a Python function to calculate fibonacci numbers' }
    ]
  })
});
```

### Python Example

```python
import requests

response = requests.post(
    'https://api.closedrouter.io/v1/chat/anthropic',
    headers={
        'Authorization': f'Bearer {api_key}',
        'Content-Type': 'application/json'
    },
    json={
        'model': 'claude-3-sonnet',
        'messages': [
            {'role': 'user', 'content': 'Analyze this dataset and provide insights'}
        ]
    }
)
```

## ðŸ“‹ Available Models Endpoint

You can retrieve all available models using the `/v1/models` endpoint:

```bash
curl https://api.closedrouter.io/v1/models \
  -H "Authorization: Bearer $CLOSEDROUTER_API_KEY"
```

This returns a list of all supported models with their capabilities and specifications.
