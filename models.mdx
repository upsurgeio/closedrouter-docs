Explore available OpenAI and Anthropic models

### Models API Standard

Our **<u>Models API</u>** makes the most important information about all LLMs freely available as soon as we confirm it.

### API Response Schema

The Models API returns a standardized JSON response format that provides comprehensive metadata for each available model. This schema is cached at the edge and designed for reliable integration for production applications.

### Root Response Object

```json
{  "data": [    /* Array of Model objects */  ]}
```

### Model Object Schema

| Field            | Type       | Description                                                                                           |
| :--------------- | :--------- | :---------------------------------------------------------------------------------------------------- |
| `id`             | `string`   | Unique model identifier used in API requests (e.g., `"openai/gpt-4o"`, `"anthropic/claude-3-sonnet"`) |
| `name`           | `string`   | Human-readable display name                                                                           |
| `provider`       | `string`   | Provider of the model: `"openai"` or `"anthropic"`                                                    |
| `description`    | `string`   | Summary of model capabilities                                                                         |
| `context_length` | `number`   | Maximum supported context window in tokens                                                            |
| `pricing`        | `Pricing`  | Cost structure (see below)                                                                            |
| `capabilities`   | `string[]` | Supported input/output types and parameters                                                           |

### Pricing Object

```typescript
{
  "prompt": string,      // Cost per input token
  "completion": string,  // Cost per output token
  "request": string      // Fixed cost per request (if any)
}
```

<Note>
  Pricing varies between OpenAI and Anthropic. Always confirm the latest rates with the `pricing` object in the `/models` response.
</Note>

### Supported Parameters

Models support OpenAI- compatible parameters where applicable:

- `tools`
- `tool_choice`
- `max_tokens`
- `temperature`
- `top_p`
- `stop`
- `frequency_penalty`
- `presence_penalty`
- `seed`

### Available Models Endpoint

```bash
curl https://api.pineapple.ai/v1/models \
  -H "Authorization: Bearer $PINEAPPLE_API_KEY"
```

This returns all supported models and their capabilities.
