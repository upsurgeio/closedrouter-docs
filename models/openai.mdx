ClosedRouter provides access to OpenAI‚Äôs GPT family under a single API.  \
Use these models for general-purpose reasoning, coding, and multimodal understanding.

The following Anthropic models are supported:

- `gpt-4o` - High capability, multimodal
- `gpt-4o-mini` - Faster, cost-efficient, multimodal

  Use the `model` field to select the exact OpenAI model you want to route to, prefixed with `openai/`

### Chat

Send a single-turn or multi-turn chat with optional system instructions.

```bash
{
    "model": "openai/gpt-3.5-turbo",
    "messages": [
        {
            "role": "system",
            "content": "You are a helpful assistant."
        },
        {
            "role": "user",
            "content": "Hello! Can you tell me a joke about programming?"
        }
    ],
    "temperature": 0.7,
    "max_tokens": 150
}
```

```bash
{
    "model": "openai/gpt-3.5-turbo",
    "messages": [
        {
            "role": "system",
            "content": "You are a helpful coding assistant."
        },
        {
            "role": "user",
            "content": "How do I reverse a string in Python?"
        },
        {
            "role": "assistant",
            "content": "You can reverse a string in Python using slicing: `reversed_string = original_string[::-1]`"
        },
        {
            "role": "user",
            "content": "Can you show me another method?"
        }
    ],
    "max_tokens": 150
}
```

### Streaming Responses

Set `"stream": true` to receive partial tokens in real time (great for chat UIs).

```bash
{
    "model": "openai/gpt-3.5-turbo",
    "messages": [
        {
            "role": "user",
            "content": "Write a short story about a robot learning to paint."
        }
    ],
    "stream": true,
    "max_tokens": 500
}
```

### Tool Calling

Let the model decide when to call your functions by providing a tool schema.

```bash
{
    "model": "openai/gpt-3.5-turbo",
    "messages": [
        {
            "role": "user",
            "content": "What is the weather in San Francisco?"
        }
    ],
    "tools": [
        {
            "type": "function",
            "function": {
                "name": "get_weather",
                "description": "Get the current weather in a location",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "location": {
                            "type": "string",
                            "description": "The city and state"
                        },
                        "unit": {
                            "type": "string",
                            "enum": ["celsius", "fahrenheit"]
                        }
                    },
                    "required": ["location"]
                }
            }
        }
    ],
    "tool_choice": "auto"
}
```

### üèó Structured Output

Enforce machine-readable JSON with `response_format`.\
Use this when you need predictable shapes for downstream systems.

```bash
{
    "model": "openai/gpt-3.5-turbo",
    "messages": [
        {
            "role": "user",
            "content": "List 3 popular programming languages with their year of creation and primary use case"
        }
    ],
    "response_format": {
        "type": "json_object"
    },
    "max_tokens": 200
}
```
